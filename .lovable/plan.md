

## Namal University GPA & CGPA Calculator

A personalized semester GPA calculator styled with Namal University's branding, featuring all 6 of your 7th semester courses.

### Design & Branding
- Clean, modern interface using Namal University's green and white color scheme
- University-inspired styling with professional academic feel
- Mobile-responsive design for easy access on any device

### Core Features

**1. Semester GPA Calculator**
- All 6 courses pre-loaded with proper credit hours:
  - Iqbaliyat (2 credit hours)
  - Parallel and Distributed Computing (3 credit hours)
  - Entrepreneurship (3 credit hours)
  - Machine Learning (3 credit hours)
  - Cyber Security (3 credit hours)
  - Final Year Project I (3 credit hours)
- Dropdown selection for each course grade using Pakistani HEC grading scale
- Real-time GPA calculation as you select grades
- Visual display of total credit hours (17) and calculated GPA

**2. Pakistani HEC Grading Scale**
- A = 4.00
- A- = 3.67
- B+ = 3.33
- B = 3.00
- B- = 2.67
- C+ = 2.33
- C = 2.00
- C- = 1.67
- D+ = 1.33
- D = 1.00
- F = 0.00

**3. CGPA Calculator**
- Section to add previous semester details (total credits earned + GPA)
- Combines current semester with previous performance
- Shows both semester GPA and cumulative CGPA

### User Experience
- Clear course cards showing course code and name
- Grade selection dropdown for each course
- Prominent GPA/CGPA display at the top
- Reset button to clear all selections
- Grade legend/reference for quick lookup
